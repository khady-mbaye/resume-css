resume sur css

definition

CSS veut dire cascading style sheet sert à modifier l’apparence de nos pages web en appliquant des styles au contenu en HTML

selecteurs , proprietes et valeurs

  • selecteurs
    Un sélecteur, tout d’abord, va servir à déterminer à quel(s) élément(s) HTML ou à quel type d’éléments on souhaite appliquer un style particulier. Si l’on souhaite appliquer un style particulier à tous nos paragraphes, par exemple, on utilisera le sélecteur « p ».
  • proprietes
    Une propriété va nous servir à modifier le style d’un élément en ciblant un critère bien particulier comme la taille d’un texte, sa police ou sa couleur par exemple.
  • valeurs
    Une valeur, enfin, va venir compléter une propriété et va en déterminer le comportement. Pour la propriété servant à changer la couleur d’un texte par exemple, la valeur va être la nouvelle couleur à appliquer.

EX: p{

      color: red;
     }
Dans cette exemple p est un selecteur simple et ceuci veut dire que nous allons appliquer un style particulier a tous les elements p.
color represente la propriete (qui sert à modifier la couleur d’un texte)
red est la valeur pour notre exemple(et c'est le caracteres que notre paragraphe va prendre)

Pour des raisons de performances du code, de clarté et d’économie de temps, je vous recommande vivement d’utiliser la dernière méthode dès que cela est possible.

div et span

En HTML, tout élément est soit de type block, soit de type inline. Par exemple, div est un élément de type block tandis que span est un élément de type inline. Les éléments de type block sont fondamentalement différents des éléments de type inline en HTML et il est essentiel de bien comprendre les différences entre ces deux types

les element de type block
  • Commence sur une nouvelle ligne
  • Occupent toute la largeur disponible
  • peuvent imbriquer les uns dans les autres et peuvent contenir des elements de type inline
EX: <p> </p> ;<h1> </h1> ; <ul> </ul> <li> </li> ; etc...
les element de type inline
  • S'insere dans une ligne ligne
  • Occupent seulement la largeur necessaire
  • peuvent imbriquer les uns dans les autres mais ne peuvent pas contenir des elements de type block
EX: <em> </em> ;<strong> </strong> ; <mark> </mark> <img> etc...

le modele des boites

Le modèle des boîtes est un concept essentiel : « tout élément d’une page est une boîte rectangulaire et peut avoir un padding, une marge et des bordures ».

Les propriétés permettant d’indiquer la longueur, la largeur, la marge, le padding et les bordures d’un élément forment ce qu’on appelle le modèle des boîtes.

proprietes valeures description
width prend des valeure en porcentage ou en pixel permet de definir la largeur d'un element
height prend des valeure en porcentage ou en pixel permet de definir la hauteur d'un element
margin prend des valeure en porcentage ou en pixel permet de definir le marge exterieur d'un element
padding prend des valeure en porcentage ou en pixel permet de definir le marge interieur d'un element
border prend des valeure en pixel permet de definir la largeur du bordure d'un element
